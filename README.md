# -2
Вектор в C++ — это стандартный шаблон обобщённого программирования, реализующий динамический массив. 2 Обеспечивает быстрое добавление новых элементов в конец и меняет свой размер при необходимости. 4

Некоторые методы работы с векторами в C++:

Добавление элементов: метод push_back() используется для добавления элемента в конец вектора. 5
Доступ к элементам: элементы вектора могут быть получены с использованием оператора [] или метода std::vector::at(). 5
Размер и ёмкость: метод std::vector::size() возвращает текущий размер вектора, а метод std::vector::capacity() — текущую ёмкость (вместимость) вектора. 5
Удаление элементов: методы std::vector::pop_back() и std::vector::erase() используются для удаления элементов из вектора. 5
